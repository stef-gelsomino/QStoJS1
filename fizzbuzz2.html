Advanced FizzBuzz

Generate a random whole number between 0 and 15. If the result is divisible by 3 alert "fizz". If the number is divisible by 5, alert "buzz". Otherwise print the number to the console.
If the random number is divisible by 3 and 5, alert "fizzbuzz". If the random number is 0, it should not return fizz, buzz or fizzbuzz.

If you open the DevTools and you take a look to the console, you can understand how it works by refreshing multiple times this page.

To see the code and understand how we solved the problem with the 0, that we previous had in the simplified example, open the "fizzbuzz2.js" file.
