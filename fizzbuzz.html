FizzBuzz

Generate a random number between 0 and 5. If the result is divisible by 3, alert "fizz". If the number is divisible by 5, alert "buzz". Otherwise print the number to the console.

If you open the DevTools and you take alook to the console, you can understand how it works by refreshing multiple times this page.

To see the code open the "fizzbuzz.js" file.
